<h1 align="center">FallMonke</h1>
<p align="center">
  <strong>FallMonke</strong> is a plugin/mod for <em>Gorilla Tag</em>.  
  It adds a whole new gamemode to Gorilla Tag, inspired by the Hex-a-gone gamemode from Fall Guys.
</p>
